# 🚌 대중교통 길찾기 서비스

한국 대중교통 경로 검색 서비스. 사용자가 직접 출발지와 목적지를 추가하고, ODsay API를 통해 최적의 대중교통 경로를 검색할 수 있습니다.

## ✨ 주요 기능

### 🔍 위치 관리
- **위치 추가**: 위치 이름으로 자동 좌표 검색 (OpenStreetMap API 활용)
- **위치 삭제**: 저장된 위치를 간편하게 삭제
- **위치 목록**: 모든 저장된 위치를 테이블로 확인

### 🚇 경로 검색
- **다중 경로 검색**: 여러 출발지와 목적지를 동시에 선택하여 비교
- **상세 경로 정보**: 이동 시간, 환승 횟수, 요금 정보 제공
- **세부 경로 보기**: 지하철/버스 노선 및 정거장 정보 상세 표시

### 💾 데이터베이스
- **SQLite 데이터베이스**: better-sqlite3를 사용하여 위치 정보 영구 저장
- **초기 데이터**: 서버 시작 시 주요 위치 9개 자동 마이그레이션

## 🛠️ 기술 스택

### Backend
- **Node.js** - JavaScript 런타임
- **Express** - 웹 프레임워크
- **better-sqlite3** - SQLite 데이터베이스
- **axios** - HTTP 클라이언트
- **cors** - CORS 미들웨어

### Frontend
- **HTML5/CSS3** - UI 구현
- **Vanilla JavaScript** - 클라이언트 로직
- **OpenStreetMap Nominatim API** - 지오코딩 (주소 → 좌표 변환)

### External APIs
- **ODsay API** - 대중교통 경로 검색

## 🎯 사용 방법

### 1. 위치 추가하기

1. **위치 이름 입력**: 예) "강남역", "홍대입구", "서울역"
2. **🔍 검색 클릭** 또는 Enter 키
3. **검색 결과 선택**: 원하는 위치를 클릭하면 자동으로 좌표가 입력됨
4. **위치 추가 버튼 클릭**

> **팁**: 더 정확한 결과를 원하시면 지역명을 함께 입력하세요 (예: "의정부 원흥역")

### 2. 경로 검색하기

1. **출발지 선택**: 체크박스에서 원하는 출발지 선택
   - "출발지 전체 선택" 버튼로 모두 선택 가능
2. **목적지 선택**: 체크박스에서 원하는 목적지 선택
   - "목적지 전체 선택" 버튼로 모두 선택 가능
3. **길찾기 시작 버튼 클릭**

### 3. 결과 확인

- **기본 정보**: 이동 시간, 환승 횟수, 요금
- **세부 경로 보기**: 각 버튼 클릭 시 상세 경로 정보 표시
  - 🚇 지하철 노선 및 정거장
  - 🚌 버스 번호 및 정거장
  - 🚶 도보 이동 거리

## 🔌 API 엔드포인트

### 위치 관리 API

| 메서드 | 엔드포인트 | 설명 |
|--------|-----------|------|
| GET | `/api/locations` | 모든 위치 조회 |
| POST | `/api/locations` | 위치 추가 (name, longitude, latitude) |
| DELETE | `/api/locations/:id` | 위치 삭제 |

### 경로 검색 API

| 메서드 | 엔드포인트 | 설명 |
|--------|-----------|------|
| POST | `/api/transit-routes` | 대중교통 경로 검색 |

**요청 예시:**
```json
{
  "origins": ["종로", "강남역"],
  "destinations": ["수원", "홍대입구"]
}
```

## ⚠️ 주의사항

1. **OpenStreetMap 검색 정확도**:
   - 전 세계 지원하지만 한국 지역의 경우 가끔 부정확할 수 있음
   - 더 정확한 결과가 필요하면 지역명 함께 입력 (예: "강남역 서울")

2. **ODsay API 제한사항**:
   - 한국 대중교통 경로만 지원
   - 일부 지하철역이나 버스정류장이 검색되지 않을 수 있음

3. **좌표 범위**:
   - 한국: 경도 124-132, 위도 33-43
   - 이 범위를 벗어나는 좌표는 "경로를 찾을 수 없음"으로 표시됨
